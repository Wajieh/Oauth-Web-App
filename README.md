# Python OAuth 2.0 From-Scratch Implementation

A clean, minimalist implementation of the OAuth 2.0 Authorization Code Flow. This project avoids high-level authentication libraries to demonstrate the underlying HTTP handshakes, state verification, and token exchange process.

## 📌 Overview

This repository serves as a learning tool for mastering the OAuth 2.0 protocol. It uses **FastAPI** for the web interface and **HTTPX** for asynchronous server-to-server communication.



## 🛠️ Tech Stack

- **Framework:** FastAPI
- **HTTP Client:** HTTPX (Async)
- **Environment Management:** Python-Dotenv
- **Server:** Uvicorn
